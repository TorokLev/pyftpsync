- Detect conflicts if both targets are modified since last sync.

- Colored output
- Interactive mode
- Renamed _pyftpsync-meta.json to .pyftpsync-meta.json
- MSI installer for MS Windows
- FTPS (TLS) support on Python 2.7/3.2+
